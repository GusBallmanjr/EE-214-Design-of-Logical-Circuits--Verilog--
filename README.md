# EE 214 — Design of Logical Circuits (Fall 2025)
# Gus Ballman

Verilog projects from WSU’s EE 214 course. This repo covers the core combinational and sequential logic designs I built and tested on the RealDigital Blackboard.

---

## What’s Inside
+ SOP/POS logic and minimized circuits  
+ Truth tables, K‑Maps, and verification  
+ Structural and behavioral Verilog modules  
+ FPGA‑ready designs 

---

## Course Focus
Hands‑on digital design from the ground up. Weekly projects introduce concepts, reinforce them through required design tasks, and add optional challenges for more advanced functionality.

---

## Outcome
Designing, simulating, and implementing combinational and sequential logic circuits in Verilog on an FPGA.
